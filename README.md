crappyblog
==========

### What is this repository for? ###

* A minimal blog platform using PHP and MySQL
* Version 1.0

### How do I get set up? ###

* Create a MySQL database 
* Rename conf_default.ini to conf.ini. This is where all your database connection info is stored.
